## 🔧 Installation

```bash

npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## 📸 Adding Photos

1. Navigate to **Scene 3: Memory Archive**
2. Click **"+ UPLOAD MEMORIES"**
3. Select multiple photos
4. Photos appear in the gallery and throughout the experience

## 🎵 Audio

- System boot sound
- Celebration sounds
- Background ambient music
- Mute/unmute toggle in top right

## 🎁 Customization

### Change Name/Age
Edit `src/scenes/Scene1ClassifiedAccess.jsx` and `src/scenes/Scene2DigitalPassport.jsx`

### Modify Messages
Update roasts in `src/scenes/Scene6RoastZone.jsx`
Update statistics in `src/scenes/Scene5AIAnalyzer.jsx`

### Color Theme
Edit Tailwind config in `tailwind.config.js`
