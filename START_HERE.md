## 🎯 How to Use

### Basic Flow
1. **Scene 1-2**: Intro & Profile (automated)
2. **Scene 3**: Upload photos
3. **Scenes 4-7**: Explore memories & have fun
4. **Scene 8**: Emotional moment
5. **Scenes 9-10**: Cake & celebration
6. **Scenes 11-13**: Grand finale

### Navigation
- Use "NEXT" buttons to proceed
- Use "PREVIOUS" buttons to go back
- Scene counter in bottom left

### Photo Upload
- Scene 3 has an upload button
- Upload 6-10 photos for best experience
- Photos appear throughout all scenes

### Audio
- Mute/unmute button in top right
- Ambient sounds throughout
- Celebration sounds in finale

## 🎨 Customization Guide

### Change Name/Birthday
Edit `src/scenes/Scene2DigitalPassport.jsx` (line 10)

### Modify Roasts
Edit `src/scenes/Scene6RoastZone.jsx` (roasts array)

### Change Final Message
Edit `src/scenes/Scene13GiftBox.jsx` (secret message text)

### Adjust Colors
Edit `tailwind.config.js` (neon color values)

### Speed Up/Slow Down
Edit `transition={{ duration: X }}` in scene files

### Remove Audio
Comment out audio code in individual scenes
