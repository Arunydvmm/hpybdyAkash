## 🔧 Common Tasks

### Upload Photos
→ Scene 3 → Click "+ UPLOAD MEMORIES"

### Change Name
→ Edit `src/scenes/Scene2DigitalPassport.jsx` (line 10)

### Add Roasts
→ Edit `src/scenes/Scene6RoastZone.jsx` (roasts array)

### Change Colors
→ Edit `tailwind.config.js` (neon colors)

### Adjust Speed
→ Find `transition={{ duration: X }}` in scenes

### Deploy
```bash
npm run build  # Creates dist/ folder
# Upload dist/ to Vercel/Netlify/GitHub Pages
```

## ⚠️ If Something Goes Wrong

| Problem | Solution |
|---------|----------|
| Port 3000 busy | Edit vite.config.js → port: 3001 |
| `npm install` fails | Delete node_modules, try again |
| Blank screen | Clear cache: Ctrl+F5 |
| Photos won't upload | Check browser console for errors |
| Slow animations | Close other apps |
