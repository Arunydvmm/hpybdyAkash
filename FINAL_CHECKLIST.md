## 🔧 Troubleshooting Prep

### If Dev Server Won't Start
- [ ] Check Node version: `node -v`
- [ ] Delete `node_modules` folder
- [ ] Delete `package-lock.json`
- [ ] Run `npm install` again
- [ ] Try `npm run dev` again

### If Photos Won't Upload
- [ ] Check browser console for errors
- [ ] Ensure photos are .jpg/.png/.gif
- [ ] Try smaller file sizes
- [ ] Clear browser cache
- [ ] Try different browser

### If Animations Stutter
- [ ] Close other applications
- [ ] Check browser task manager
- [ ] Reduce particle effects (code)
- [ ] Try simpler browser (Chrome)
- [ ] Test on different device

### If Port 3000 is Busy
- [ ] Kill process: `lsof -i :3000` (Mac/Linux)
- [ ] Or edit `vite.config.js` to use port 3001
- [ ] Restart VS Code
